Oops

Object Oriented programming(Oop) is a progarmming paradigm that uses objects,which can contain data in the form of fields (often known as properties or attibute) and code in the form of procedures(often known as methods).Javascript is versatile language that supports OOP principles throgh its prototype based inheritance system.

A class in Javascript is a blueprint for creating objects.it defines a set of properties and methods that objects created from the class will have.
